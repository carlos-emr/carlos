## 2024-06-25 - ARIA attributes and href validity in dynamically constructed HTML
**Learning:** In the CARLOS EMR app, dynamic HTML string builders are frequently used in JSPs to render tables and interfaces (e.g., `admin/jobs.jsp`). Elements acting as buttons within these strings often rely on `onclick` handlers but lack a proper `href` attribute (often set to the invalid `javascript:void();` or omitted), rendering them unreachable via keyboard tabbing. Decorative icons within these dynamic elements are frequently read by screen readers if not explicitly hidden.
**Action:** When refactoring dynamic HTML generation, always ensure anchor tags acting as buttons have a valid `href="javascript:void(0);"` for focusability, add `aria-label`s for icon-only actions, and set `aria-hidden="true"` on decorative inner icon tags (e.g., `<i class="fa-solid fa-calendar">`) to provide a consistent, accessible experience.
