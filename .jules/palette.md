## 2024-06-03 - Improved Keyboard Focus and ARIA Labels in Dynamic Tables
**Learning:** When building HTML strings dynamically in JavaScript (e.g., for data tables), anchor tags (`<a>`) used as buttons MUST include a valid `href` attribute to be keyboard focusable via tabbing. Avoid `href="javascript:void();"` (which can cause console errors or invalid behavior) and use `href="javascript:void(0);"`. Additionally, icon-only buttons need explicit `aria-label` attributes to provide context to screen readers, and purely decorative icons should be hidden using `aria-hidden="true"`.
**Action:** Always ensure any `<a>` tag with an `onclick` handler has `href="javascript:void(0);"` if it doesn't navigate. Add `aria-label` to icon-only links and `aria-hidden="true"` to inner icon spans/tags.
