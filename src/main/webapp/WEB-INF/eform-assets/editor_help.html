Templates

Use the template dropdown to load a pre-formatted letter template. Template placeholders (e.g., ##patient_name##) are automatically replaced with patient data from the chart.

Tips

  • Always click Submit to save your letter to the patient's chart.
  • The Print button prints and saves simultaneously.
  • Use Export to save the letter as a standalone .rtl file.
  • Custom templates (.rtl files) can be uploaded to the eForm images directory by an administrator.
